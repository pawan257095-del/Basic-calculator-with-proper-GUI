# 🧮 Basic Calculator (GUI)

A simple and user-friendly **Basic Calculator** built with a graphical user interface (GUI). This application performs standard arithmetic operations with an intuitive layout, making it perfect for beginners and everyday use.

---

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear / Reset functionality
* 🖱️ Interactive buttons with smooth GUI experience
* 📟 Real-time input and result display

---

## 🖥️ GUI Overview

The calculator comes with a clean and minimal interface:

* A display screen to show inputs and results
* Number buttons (0–9)
* Operation buttons (+, −, ×, ÷)
* Equals (=) button to calculate results
* Clear (C) button to reset
